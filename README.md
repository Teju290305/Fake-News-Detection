# Fake-News-Detection
Fake News Detection Project

Project Title:
Fake News Detection using Machine Learning

Description:
This project is developed to detect whether a news article is real or fake using Machine Learning.

The system analyzes the news text and predicts the result.

Technology Used:
- Python
- Machine Learning
- Random Forest Algorithm

Libraries Used:
- Pandas
- Sklearn
- Numpy
- Matplotlib

Dataset:
The dataset contains news text and labels.

Label:
1 = Real News
0 = Fake News

Files in Project:

fake_news_detection.py
Python source code

news_093.csv
Dataset file

result.png
Output screenshot

Accuracy:
The model achieved accuracy of 93%.

How to Run Project:

1 Install Python

2 Install Libraries

pip install pandas
pip install sklearn
pip install matplotlib

3 Run the code

python fake_news_detection.py

Output Example:

Accuracy: 0.93

Conclusion:
The project successfully detects fake news using machine learning.
